Implement M One Payment gateway your website, you can add Buy now button in any posts or pages of your website.

== Description ==

This M1Pay plugin will add a new block type to WordPress default editor called Buy now Button, so that you can easily use the block anywhere in the posts or pages.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/m1pay-pay-button` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Plugins->M1Pay Buy now Button->Settings to configure the plugin.

== Frequently Asked Questions ==

= My M One Buy Now button plugin does not redirect to M One Gateway website =

Check you have entered the correct authentication information into the setting section.

= After payment is done or canceled the M One gateway does not redirect to the correct address on my website =

You have to send the correct URL to the M One support team to set as redirect address. You can pick the URL from plugin setting page.
